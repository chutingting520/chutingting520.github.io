mysql的学习

重复安装了很多遍，网上教程与安装时的顺序不一样，通过remove移除再下载，多尝试几遍就知道如何安装了。

后面直接按照phpstudy就行了，集成的。

1、创建数据库

——create database 数据库名称；（注意有分号）
此方法不能立马刷新，可以在左下角点击右键Refresh all 来刷新。

2、创建表、删除表
——create table 表名（字段1 字段类型，字段2 字段类型，字段3 字段类型）；后面还能继续加
eg：create table Phone_table(pid INT,name CHAR(20),price INT);
*注意创表的时候需使用use指明你要使用的数据库，use +数据库名（这里的分号是唯一可省略的)
删除表：drop table 表名;

3、修改表名
——alter table 表名 rename to 新表名；或
——rename table 表名 to 新名；

4、为数据表增加、修改、删除字段
——alter table 表名 add 字段名 字段类型;
添加多个时记得加括号，每个字段之间逗号隔开。

——alter table 表名 change 旧字段名 新字段名 字段类型。
